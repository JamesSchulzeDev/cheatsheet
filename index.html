Cheat Sheet

HTML Tags

Syntax Name Description
<H1> Main Heading Heading of the page. Only use 1 H1 per page
<H2> Sub Heading Sub headings after H1.
<H3> Smaller Headings Headings range from H1 to H6.
<p> Paragraph Where you add text into for reading
<div> Div Block A block element
<img> Image Use to add an image in a web page

Table Tags

Syntax Name Description
<table> Table Element for HTML tables
<thead> Table head The column headers in a table
<tbody> Table Body The rows containing the actual table data
<tr> Table Row Table row container
<td> Table Data The data inside the table row
<tfoot> Table Foot The set of rows defining the footer in a table

Table Attributes

Syntax Name Description
<colspan> Column Span How many coloumns a td element should span
<rowspan> Row Span How many rows a td element should span

Form

Syntax Name Description
<form> Form element is used to collect and send information to an external source.
<input> Checkbox Type attribute will render a single checkbox item.
<textarea> Text Area element is used when creating a text-box for multi-line input

Lists

Syntax Name Description
<ul> Unordered Lists Each list item starts with a dot
<ol> Ordered List Each list item starts with a number
<li> List Item The data that goes into each list line
